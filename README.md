# BPSCM WebServiceAPI Library (金財通WebServiceAPI 串接模組)

### Support (可用功能)
1. Upload InvoiceOrder (開立發票)
2. Upload AllowanceOrder (開立折讓單)

### Install (安裝)
    composer require linmasahiro/bpscm
